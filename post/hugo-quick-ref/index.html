    Hugo Quick Reference

    Install Hugo

    install hugo via chocolate

    choco install hugo -confirm

    install Sass/SCSS version

    choco install hugo-extended -confirm

    Hugo Cmd

    new site

    hugo new site <FolderName>

    new post

    hugo new post/hello-world.md

    preview site

    hugo server -D

    build site

    hugo
